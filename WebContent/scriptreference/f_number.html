number

ホーム

number of 〜s

オブジェクトやチャンクの数を返します。これとよく似ているnumberプロパティはオブジェクトの番号を表します。 

on mouseUp
  --オブジェクト
  put number of cd btns into BTNS
  put number of cd flds into FLDS
  put number of cards into CARDS
  put number of bgs into BKGNDS
  --チャンク
  put number of chars in "abc" into CHARS
  put number of items in "a,b,c" into ITEMS
  put number of lines in "abc" into LINES
  put number of words in "a b c" into WORDS
  --他
  put number of menus into MENUS
  put number of menuitems of me "Stack" into MENUITEMS
end mouseUp
