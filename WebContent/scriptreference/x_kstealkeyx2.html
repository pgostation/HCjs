KStealKeyX2

スクリプト中で現在押されているキー状態を取得する拡張機能です。主にゲームスタックで利用されます。iPadでもバーチャルキーパッドが表示され、キー入力できます。 

on mouseUp
  repeat
    put KStealKeyX2("adws") into key
    if char 1 of key >= 1 then
      -- 左へ移動
    end if
    if char 2 of key >= 1 then
      -- 右へ移動
    end if
    if char 3 of key >= 1 then
      -- 上へ移動
    end if
    if char 4 of key >= 1 then
      -- 下へ移動
    end if
  end repeat
end mouseUp
