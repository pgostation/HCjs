HCjs内蔵のコマンドと関数

HCjsでは内蔵のコマンドと関数が用意されています。一覧と説明はスクリプトリファレンスを参照してください。

内蔵コマンドanswerの使用例

on mouseUp
    answer "hello"
end mouseUp

内蔵関数の使用例

on mouseUp
    put abs(-2)
    put abs of -2
    -- どちらの書き方でも同じように内蔵のabs関数が呼び出されます。(HyperCardと異なる)
end mouseUp

on mouseUp
    put clickChunk()
    put the clickChunk
end mouseUp

スクリプトで作成するユーザ定義のコマンドと関数

スクリプトでユーザで定義するコマンドと関数の二種類のハンドラが作成できます。コマンドはメッセージを受け取るため、関数はスクリプトから呼ばれて値を返すために使います。

ユーザ定義コマンドの例

on mouseUp
    set icon of me to 1001
end mouseUp

ユーザ定義関数の例

function goukei a,b
    return a+b
end goukei

内蔵コマンド/関数のユーザ定義コマンド/関数による上書きはできません

内部で定義されているコマンド/関数は内蔵コマンド/関数が実行されます。ユーザで同じ名前のコマンド/関数を定義しても使えません。

内蔵コマンド/関数の引数は最大8個
