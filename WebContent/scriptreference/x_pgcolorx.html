PgColorX

カラー化を行う拡張機能です。HyperCardからカラースタックを変換したときに使用されます。 

on mouseUp
  addColor "new", port, size/picture id/picture name
  addColor "copyBits", 転送元port, 転送先port, 転送元rect, 転送先topleft/rect, mode
  addColor "tiling", 転送元port, 転送先port, 転送元rect, 転送先rect, mode
  addColor "pict", port, picture id/picture name, topleft/rect
  addColor "string", port, text, color, topleft
  addColor "kill", port
end mouseUp
