ask 表示文字列

ask 表示文字列 with デフォルト文字列

文字入力のためのダイアログを表示します。ユーザが入力した文字列はitに入ります。 

on mouseUp
  ask "名前を入力してください"
  if it is not empty then
    put it into cd fld "名前"
  end if
end mouseUp
