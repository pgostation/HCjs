cantModify

ホーム

cantModify of [スタック]

スタックの変更不可設定をtrue/falseで表します。変更不可に設定されている場合、スタックを変更しても保存されません。 

-- idleでアニメーションしている間保存処理を行わない
on openCard
  set cantModify of this stack to true
end openCard

on idle
  set loc of cd btn 1 to the mouseLoc
end idle

on closeCard
  set cantModify of this stack to false
end closeCard
