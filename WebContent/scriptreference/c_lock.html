lock

ホーム

lock screen

画面の更新を停止します。unlock screenで画面を更新します。 

on mouseUp
  lock screen
  hide me --この時点ではまだ画面に見えている
  wait 1 sec
  unlock screen --この時点で画面が更新される
end mouseUp

lock messages

カード移動によるメッセージ送信(openCardやcloseCard)を行わなくします。unlock messagesで、カード移動時にメッセージ送信を行うようになります。 

on mouseUp
  lock messages
  go next
  unlock messages
end mouseUp

lock recent

カード移動の履歴(リーセント)の記録を停止します。unlock recentで記録するようになります。 

on mouseUp
  lock recent
  go next
  unlock recent
end mouseUp
