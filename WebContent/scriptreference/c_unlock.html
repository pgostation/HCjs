unlock

ホーム

unlock screen

lock screenで停止した画面を更新します。 

on mouseUp
  lock screen
  hide me --この時点ではまだ画面に見えている
  wait 1 sec
  unlock screen --この時点で画面が更新される
end mouseUp

unlock screen with 視覚効果

lock screenで停止した画面から視覚効果付きで現在の画面を表示します。視覚効果はvisualコマンドと同じです。 

on mouseUp
  lock screen
  hide me
  unlock screen with dissolve
end mouseUp

unlock messages

lock messagesで停止したカード移動時のメッセージ送信を行うようにします。 

on mouseUp
  lock messages
  go next
  unlock messages
end mouseUp

unlock recent

lock recentで停止したリーセントの記録を再開します。 

on mouseUp
  lock recent
  go next
  unlock recent
end mouseUp
