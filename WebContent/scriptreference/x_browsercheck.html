BrowserCheck

ホーム

BrowserCheck(string)

Webブラウザに関する情報を取得する拡張機能です。

  • userAgent: ブラウザの種類を判別するためのユーザーエージェント文字列を取得します。
  • webAudioAPI: ブラウザがWebAudioAPIに対応しているかどうかをtrue/falseで表します。
on mouseUp
  if "Chrome" is in BrowserCheck("userAgent") then
    -- 処理
  end if
  if BrowserCheck("webAudioAPI") is false then
    -- 処理
  end if
end mouseUp
